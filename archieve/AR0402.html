디모데성경연구원의 ppt는 폰트저작권에 위배되지 않는 무료폰트를 사용하고 있습니다.
아래의 버튼을 클릭하여 폰트를 다운 받고, 설치하셔야 ppt를 열었을 때 폰트가 깨지지 않습니다.
폰트가 설치되지 않은 컴퓨터의 경우, PPT를 읽기전용으로 불러와야 글자 배열이 깨지지 않습니다.

제공한 PPT는 서울체/나눔체/다음체로 제작되었으니 아래의 "첨부파일"을 다운로드 받아 설치하시기 바랍니다.

설치 방법
1. 첨부파일을 다운로드 한다.
2. 받은 파일의 압축을 푼다.
3. 폴더 안의 파일을 전체 복사한다. (Ctrl+C)
4. 폴더를 새로 연다.(폰트가 들어있는 폴더)
(파일 탐색기 -> C드라이브 -> Windows 폴더 -> Fonts 폴더)
5. 아까 3번에서 복사한 파일을 Fonts 폴더에 붙여 넣기 한다. (Ctrl+V)
6. 그러면 복사하고 있다는 작은 창이 떴다 사라진다.
7. 컴퓨터를 재부팅 후 PPT를 연다.
